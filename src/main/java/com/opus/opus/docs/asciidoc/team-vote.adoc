== API 목록

link:./opus.html[API 목록으로 돌아가기]

투표는 `PUT`/`DELETE` 두 엔드포인트로 분리되어 있으며, 두 메서드 모두 멱등합니다.

* `PUT /teams/{teamId}/votes` → 투표 등록 (이미 등록된 경우 NO-OP, 200 OK)
* `DELETE /teams/{teamId}/votes` → 투표 취소 (이미 취소된 경우 NO-OP, 200 OK)

NOTE: 두 메서드 모두 응답 본문에 사용자의 남은 투표 가능 횟수와 대회 최대 투표 허용 개수를 포함합니다.

NOTE: 투표 등록 시 대회 최대 투표 수 초과 시 400 에러가 발생합니다.

NOTE: 투표 기간 중에만 호출 가능하며, 투표 기간이 아닐 때 호출하면 400 에러가 발생합니다.
