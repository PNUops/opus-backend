== 공개 범위 (SubmissionVisibility)

제출 항목의 `visibility` 값에 따라 제출물 열람 권한이 달라집니다.

[cols="1,4"]
|===
| 값 | 열람 가능 대상

| `TEAM`
| 제출 팀의 팀원, 관리자

| `STAFF`
| 제출 팀의 팀원, 해당 팀에 지정된 지도교수, 해당 팀에 지정된 외부멘토, 관리자

| `MEMBER`
| 로그인한 모든 회원

| `PUBLIC`
| 전체 (비회원도 열람 가능)
|===
